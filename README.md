# 보물찾기 게임 (교내 게임잼)

> 신호기로 묻힌 보물의 위치를 추적해
> 제한 시간 내에 찾아내는 미니게임

## 개발 배경
교내 게임잼 참가작입니다.
1박 2일 동안 2인 팀으로 개발했으며
클라이언트 코드는 대부분 혼자 담당했습니다.

## 게임 소개
그리드 형태의 맵에서 신호기를 작동시켜
묻혀있는 보물과의 거리를 파악하고
제한 시간 내에 보물을 찾아야 합니다.

## 기술 스택
Unity / C#

## 담당 구현
그리드 기반 맵 시스템
신호기 거리 감지 로직
제한 시간 UI 및 게임 흐름 관리
